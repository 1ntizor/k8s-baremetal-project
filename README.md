# Kubernetes Baremetal Project 🏗️

[Russian version below]

This project demonstrates advanced Nginx configuration in a Baremetal Kubernetes cluster. It covers lifecycle management, security, and scheduling.

## 📁 Project Structure:
* **app/**: Nginx setup with Probes, Affinity, and Tolerations. 🚀
* **database/**: PostgreSQL configurations. 🐘
* **infrastructure/**: Base cluster settings. 🛠️
* **rbac/**: Role-Based Access Control settings. 🛡️

## 🔧 Features:
* **Self-healing**: Configured Liveness and Startup probes. 🩺
* **Smart Scheduling**: Node Affinity and Tolerations for precise placement. 🧲
* **Zero Downtime**: Rolling updates for seamless deployment. 🔄

# Проект Kubernetes на Baremetal 🏗️

Этот проект демонстрирует продвинутую настройку Nginx в кластере Kubernetes на "голом железе". Включены аспекты управления жизненным циклом, безопасности и планирования.

## 📁 Структура проекта:
* **app/**: Настройка Nginx с пробами, Affinity и Tolerations. 🚀
* **database/**: Конфигурации для PostgreSQL. 🐘
* **infrastructure/**: Базовые настройки кластера. 🛠️
* **rbac/**: Настройки ролевого доступа (RBAC). 🛡️

## 🔧 Что реализовано:
* **Самолечение**: Настроены Liveness и Startup пробы. 🩺
* **Умное планирование**: Использование Node Affinity и Tolerations. 🧲
* **Отказоустойчивость**: Обновления через RollingUpdate без прерывания работы. 🔄
